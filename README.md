# Landing Page

A landing page built as part of [The Odin Project](https://www.theodinproject.com/) Foundations curriculum.

## [Live Demo](https://ommagatto.github.io/landing-page/)

## This project focuses on practicing the following skills:

- Structuring a complete web page with semantic HTML elements (`<header>`, `<section>`, `<footer>`)
- Building layouts with CSS Flexbox - aligning, centering, and distributing space across the page
- Applying the CSS box model (margins, padding, borders) to control spacing and structure
